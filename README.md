# Antibioticafinder
Diese App hilft dir dabei, das passende Antibiotikum zu finden.
Du gibst einfach die Symptome und die Art der Bakterien ein.

Die App analysiert deine Angaben und zeigt dir, welches Antibiotikum am besten geeignet ist. So erhältst du schnell eine hilfreiche Orientierung.
